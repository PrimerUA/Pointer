MainScreen

1) get: list[Point] - получение  списка координат точек от сервера для нанесения на карту (параметры from, length)

PointDetailsScreen

2.1) get: Point - получение деталей выбранной точки (параметр point_id)
2.2) post:согласие на встречу /!/отправка пуш уведомления партнеру о согласии

UserProfileScreen

3) get: UserInfo - получение полного профиля пользователя (параметр user_id)

NewDescriptionScreen

4) post: Point - создание новой точки встречи (параметр point_id)

ConfirmPointScreen

5.1) get: ConfirmationCode - код для подтверждения сделки (параметр user_id)
5.2) post: ConfirmationCode - проверка совпадения кодов

AllPointsScreen

6) get: list[PointView] - получение списка сделок с основной информацией о них (параметры from, length, type = 0,1)

RewardsScreen

7) post: list[Rewards] - отправка наград, которые лайкнул пользователь своему партнеру (параметр user_id)

CurrentPointScreen

8) post: отказ от встречи /!/отправка пуш уведомления партнеру об отказе
